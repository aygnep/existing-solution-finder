我要实现一个名为 tool-resolver 的 TypeScript CLI 项目。

目标：
输入开发者的问题描述或错误日志，输出可能解决问题的已有工具、GitHub 项目、issue 或 workaround。

本阶段只实现 MVP：
1. CLI 接收一段文本输入
2. problem-parser 抽取软件名、错误关键词、协议/API、问题类型
3. query-generator 生成多组搜索 query
4. scorer 对 mock candidates 打分
5. 输出 ranked candidates

限制：
- 不要实现 Web UI
- 不要接数据库
- 不要自动安装工具
- 不要执行任何外部代码
- GitHub API 可以先设计接口，但第一步用 mock provider
- 必须写测试
- 必须保持 core 模块为纯函数

请先创建项目结构和最小可运行版本。
